# Claude Code Codex Strongest configuration for claude-code-codex-strongest

This directory is a safe template for `~/.codex` and the Codex-side companion to Claude Code Codex Strongest.

- `AGENTS.md` gives Claude Code Codex Strongest the same high-level workflow and safety rules as this repository.
- `config.toml` keeps Claude Code Codex Strongest configuration minimal and local-user-editable.
- `.gitignore` prevents auth, tokens, sessions, logs, cache, and other runtime state from being copied into source control.

Do not place API keys, auth files, session transcripts, logs, caches, or other runtime state in this template directory.
